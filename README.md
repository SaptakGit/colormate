# ColorMate – Professional Painting Services in Kolkata

A modern, premium React-based website for ColorMate, specializing in professional interior, exterior, texture, and waterproofing services in Kolkata.

## 🚀 Tech Stack

- **Framework**: [React](https://react.dev/) (Vite)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Typography**: Playfair Display (Headings), DM Sans (Body)
- **Icons**: Emoji & Custom SVG

## ✨ Key Features

- **Interactive Cost Estimator**: Real-time cost estimation based on area, coats, and paint quality.
- **Visual Swatch Previewer**: Dynamic color previewing on the hero section.
- **Filterable Project Gallery**: Categorized portfolio for Interior, Exterior, Textures, and Waterproofing.
- **Service Areas Hub**: Location-specific details for Behala, Salt Lake, Ballygunge, and New Town.
- **Searchable Blog**: Knowledge base for painting tips and monsoon-proofing guides.
- **Responsive Navigation**: Mobile-first header, sticky navigation, and floating WhatsApp support.
- **Lead Generation**: Inquiry form with success state and WhatsApp integration.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SaptakGit/colormate.git
   cd colormate
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Reusable UI components (Calculator, Gallery, Hero, etc.)
- `src/App.jsx`: Main application controller and routing.
- `src/index.css`: Global styles and Tailwind 4 theme configuration.
- `public/`: Static assets.
